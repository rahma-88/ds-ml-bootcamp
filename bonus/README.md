# Bonus Guides

Optional guides for after you finish the core 7-lesson bootcamp. Each one links to the next at the bottom, so you can also just start with the first guide and follow the "Proceed to" links through to the end.

**Note on filenames:** the numeric prefixes don't run in reading order (a rename left them out of sync with the actual flow) — use the list below, not alphabetical file order, to read them in sequence.

## Reading Order

1. [Intro to Deep Learning](02-intro-to-deep-learning.md) — what Deep Learning is, why it took off, and the three core architectures (MLPs, CNNs, Transformers).
2. [Math for Data Science & ML](01-math-for-ds-and-ml.md) — the statistics, probability, linear algebra, and calculus already hiding inside every model you built in this course.
3. [AI & Generative AI](03-ai-and-generative-ai.md) — AI vs ML vs Deep Learning, LLMs, embeddings, vector databases, RAG, fine-tuning vs prompting, and AI agents.
4. [Career Path in ML](05-career-path-in-ml.md) — the main job titles in data and ML, what each actually does, and next steps for each path, plus recommended resources.
5. [Building a Portfolio](06-building-a-portfolio.md) — GitHub, README writing, Kaggle, LinkedIn, resume, personal website, and networking.
6. [AI Ethics & Responsible AI](04-ai-ethics.md) — bias, fairness, privacy, hallucinations, copyright, and security.
7. [Final Instructor Advice](07-final-instructor-advice.md) — closing thoughts: common beginner mistakes, why projects beat certificates, and what to learn next.
