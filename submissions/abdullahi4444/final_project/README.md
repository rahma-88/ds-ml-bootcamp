# Final Project – Capstone Assignment

## Overview
This folder contains the complete capstone project for the DS/ML bootcamp. It includes:
- **project-proposal.pdf** – approved project plan (required).
- **README.md** – optional short summary and link to the public GitHub repository.

## How to Use
1. **Read the proposal** (`project-proposal.md`) to understand the dataset, problem statement, and chosen algorithms.
2. **Implement the project** in your own GitHub repository (code, API, trained models, and documentation).
3. **Optional**: Add a short `README.md` with a project overview and a link to the GitHub repo for quick reference.


*For any questions about scope, dataset choice, or technical blockers, reach out before the respective deadlines.*
