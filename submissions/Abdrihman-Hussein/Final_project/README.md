# Flood Probability Prediction API

This project predicts the probability of flooding in a region based on 20 environmental, infrastructural, and socio-economic features (Monsoon Intensity, Urbanization, River Management, Deforestation, Dams Quality, Coastal Vulnerability, etc.). I trained and compared three regression models — Linear Regression, Random Forest, and Gradient Boosting — and deployed the best-performing one as a FastAPI service.
