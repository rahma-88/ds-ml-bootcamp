# AI Readiness Among Students: A Dataset for Machine Learning

## Abstract
Artificial Intelligence (AI) is transforming education, research, and industry worldwide. This project presents a dataset collected from university students to assess their readiness for AI technologies.

## 1. Introduction
Data Science involves collecting, analyzing, and interpreting data. Machine Learning is a branch of Data Science that enables computers to learn patterns and make predictions.

## 2. Dataset Collection Method
The dataset was collected through a questionnaire distributed to university students.

- Number of Rows: 50
- Number of Columns: 7

## 3. Features and Label

### Features (X)
- Age
- Department
- Programming_Experience_Years
- AI_Tools_Used
- Weekly_AI_Usage_Hours
- AI_Training_Attended

### Label (y)
- AI_Readiness_Level (Low, Medium, High)

## 4. Dataset Structure

| Property | Value |
|----------|--------|
| Rows | 50 |
| Columns | 7 |
| Features | 6 |
| Label | 1 |

## 5. Data Quality Issues
- Missing values
- Typographical errors
- Duplicate records
- Inconsistent formatting
- Class imbalance

## 6. Learning Type
This is a Supervised Learning problem because it contains a target variable (AI_Readiness_Level).

### Machine Learning Task
Classification

## 7. Use Case
- Predicting students' AI readiness levels.
- Identifying students who need AI training.
- Supporting curriculum development.

## 9. Conclusion
The dataset provides valuable insights into students' preparedness for Artificial Intelligence and can be used to build classification models for educational planning and AI training initiatives.
