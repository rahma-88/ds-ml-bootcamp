## Car Price Prediction Using Linear Regression and Random Forest

### 1. What Did I Implement?

In this assignment, I built two machine learning regression models to predict used car prices using the cleaned dataset from Assignment Three. I first loaded the cleaned dataset, selected **Price** as the target variable, and used all remaining columns except **LogPrice** as input features. The dataset was then divided into training (80%) and testing (20%) sets using `random_state = 42`.

I trained two different regression algorithms: **Linear Regression** and **Random Forest Regressor**. After training both models, I evaluated their performance using four evaluation metrics: R², MAE, MSE, and RMSE. Finally, I performed a sanity check by selecting one record from the test set and comparing the actual car price with the predictions generated by both models.

---

## 2. Comparison of Models

The sanity check showed that the actual price of the selected car was **$7,009**. Linear Regression predicted **$5,374**, while Random Forest predicted **$6,271**. Although neither model predicted the exact price, the Random Forest prediction was closer to the actual value.

However, when comparing the overall evaluation metrics, the results were mixed. Linear Regression achieved a higher **R² score (0.416)**, meaning it explained more of the variation in car prices across the entire test dataset. On the other hand, Random Forest achieved a lower **MAE (1,199)** than Linear Regression (**1,474**), indicating that its average prediction error was smaller.

---

## 3. Understanding Random Forest

Random Forest is an ensemble machine learning algorithm that combines many decision trees to make predictions. Each decision tree is trained using a random sample of the training data and a random subset of features. Instead of relying on a single tree, the model averages the predictions from all trees to produce the final prediction.

This ensemble approach helps reduce variance and usually improves prediction stability. Random Forest is especially useful when the relationship between features and the target variable is complex or nonlinear.

---

## 4. Metrics Discussion

The evaluation metrics revealed that neither model was better in every aspect. Linear Regression produced a higher **R² score (0.416)** than Random Forest (**0.307**), suggesting that it explained a greater proportion of the variation in the target variable.

On the other hand, Random Forest achieved a lower **MAE (1,199)** than Linear Regression (**1,474**), meaning its average prediction error was smaller. However, Linear Regression also had lower **MSE (3,888,096)** and **RMSE (1,972)** compared to Random Forest (**4,612,325** and **2,148**, respectively). These results indicate that Linear Regression made fewer large prediction errors overall.

Therefore, the evaluation metrics suggest that Linear Regression performed better overall on this dataset, while Random Forest produced a more accurate prediction for the selected sample in the sanity check.
