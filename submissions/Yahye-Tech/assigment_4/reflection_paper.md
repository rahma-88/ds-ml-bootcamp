Assignment Four: Regression – Theory and Practice

1. What Did You Implement?

In this assignment, I developed two machine learning regression models to predict car prices using the cleaned dataset from Assignment Three. The dataset was first loaded into a Jupyter Notebook, where the target variable was defined as Price, while all remaining variables except LogPrice were used as input features. The data was then divided into training and testing sets using an 80/20 split with a random state of 42 to ensure reproducibility.

Two regression algorithms were trained: Linear Regression and Random Forest Regression. After training, both models were evaluated using four common regression metrics: R², Mean Absolute Error (MAE), Mean Squared Error (MSE), and Root Mean Squared Error (RMSE). Finally, a sanity check was performed by comparing the actual price of one test sample with the predictions produced by both models.

---

2. Comparison of Models

The predictions generated by the two models were not exactly the same. Linear Regression produced predictions based on a linear relationship between the input variables and the target, while Random Forest was able to capture more complex relationships within the data.

During the sanity check, the Random Forest prediction was generally closer to the actual car price than the Linear Regression prediction. This suggests that Random Forest was able to learn the underlying patterns in the dataset more effectively. Since car prices are influenced by many interacting factors, Random Forest often provides more realistic predictions.

---

3. Understanding Random Forest

Random Forest is an ensemble machine learning algorithm that combines the predictions of many decision trees. Each tree is trained using a random sample of the training data and a random subset of features. After all trees make their predictions, the final prediction is calculated by averaging the outputs of all trees.

This approach reduces the impact of individual errors made by single decision trees and usually produces more accurate and stable predictions. Random Forest is also less likely to overfit than a single decision tree because it relies on the collective decision of many trees.

4. Metrics Discussion

The performance of both models was evaluated using R², MAE, MSE, and RMSE.

The model with the higher R² value explained more variation in the car prices and therefore provided better predictive performance. Lower values of MAE, MSE, and RMSE indicated that the model made smaller prediction errors.

In this assignment, Random Forest achieved better performance than Linear Regression because it produced a higher R² score and lower error values. These results indicate that Random Forest is better suited for datasets where the relationships between variables are complex and nonlinear.
