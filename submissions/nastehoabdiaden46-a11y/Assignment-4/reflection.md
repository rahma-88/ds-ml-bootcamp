Reflection Paper: Car Price Prediction

What I Implemented

In this assignment, I implemented two Machine Learning models to predict car prices using a cleaned car dataset from Assignment Three. The models used were Linear Regression and Random Forest Regression. The dataset was first prepared by selecting the target variable and input features. The target variable was the car price, while the remaining columns were used as input features.

The dataset was divided into training and testing sets using an 80/20 split. The training data was used to train both models, while the testing data was used to evaluate their performance. After training, I calculated evaluation metrics including R², MAE, MSE, and RMSE to compare the models.

Comparison of Models

The predictions generated by the two models were different during the sanity check. Linear Regression produced predictions based on a linear relationship between variables, while Random Forest captured more complex relationships within the data.

Random Forest generally produced more realistic predictions because it combined results from multiple decision trees instead of relying on a single mathematical equation. This allowed the model to identify patterns that Linear Regression could not capture.

Understanding Random Forest

Random Forest is an ensemble Machine Learning algorithm that combines many decision trees to make predictions. Instead of relying on one tree, it creates several trees using random subsets of data and features. Each tree produces its own prediction, and the final prediction is obtained by averaging all predictions.

This method improves prediction accuracy and reduces the risk of overfitting. Because different trees learn different patterns, Random Forest usually performs better than a single model.

Metrics Discussion

Based on the evaluation results, Random Forest achieved a better R² score and lower error values such as MAE and RMSE compared to Linear Regression. A higher R² value means the model explained more variation in car prices, while lower error values indicate more accurate predictions.

Linear Regression performed reasonably well, but it struggled with complex patterns and nonlinear relationships. Random Forest was stronger because it handled different relationships between features more effectively.

My Findings

From this experiment, I found that both models were useful for car price prediction, but Random Forest provided better overall performance. It generated more accurate predictions and achieved lower error rates.

I would prefer Random Forest for future car price prediction tasks because real-world datasets often contain complex relationships that cannot be represented by a simple linear equation. Although Linear Regression is easier to understand and faster to train, Random Forest offers greater flexibility and improved prediction quality.
